# 🧑‍💼 Full Stack Job Portal Website

A complete job listing and job application portal built using the MERN stack (MongoDB, Express.js, React.js, Node.js), featuring:

- 🔐 Clerk Authentication (Job Seekers)
- 🧾 Resume Uploads & Profile Management
- 🧑‍💻 Recruiter Dashboard for Job Postings
- 📂 Application Management (Accept/Reject/Download Resume)
- 📊 Sentry Integration for Error & Performance Monitoring
- ☁️ Deployment via Vercel

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- Clerk (Authentication)
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- Multer (Resume Uploads)
- Sentry (Monitoring & Performance)

### Deployment
- Frontend: Vercel
- Backend: Railway / Render / Cyclic
- Database: MongoDB Atlas

---

## 📁 Features

### 👨‍🎓 Job Seeker
- Sign up & Login using Clerk
- View jobs by category, location or keyword
- Apply to jobs with uploaded resume
- Track applications in user dashboard

### 🧑‍💼 Recruiter
- Secure Recruiter Login
- Post new job listings
- View and manage all posted jobs
- View applicants for a job
- Accept / Reject applications
- View and download applicant resumes

### ⚙️ Admin (Optional Feature)
- View overall stats (jobs, users, applications)
- Control visibility of job posts
- Moderate user content
